# Result Components

These are logical components to reuse across microtools.

## ScoreCircle

Shows a 0-100 score and a label.

Bands:

- red: below 50;
- amber: 50-70;
- green: 70+.

## IssueCard

Fields:

- rank;
- title;
- severity;
- impact estimate;
- root cause;
- fix;
- context-specific note.

## WeekPlanAccordion

Use for 7-day, 30-day, or 90-day plans.

## QuickWinsList

3 actions the user can take today or this week.

## TemplateBlock

Copyable output such as:

- SLA template;
- email rewrite;
- CV summary;
- offer rewrite;
- follow-up script.

## HonestSignal

Always show caveats:

- missing data;
- contradictory answers;
- privacy warning;
- "directional, not final advice".

## ShareBar

Actions:

- copy result link;
- copy report;
- download text/markdown/PDF;
- send to email/Telegram if captured.

## PaidCTACard

Fields:

- headline;
- credibility line;
- deliverables;
- price range;
- timeline;
- Telegram/booking link.

## PrivacyNote

Tell users:

- what is stored;
- what is not needed;
- what sensitive data to remove;
- how to request manual help.
