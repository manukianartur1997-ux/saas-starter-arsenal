# Microtool Kit

Reusable v1 patterns for AI diagnostic tools.

This kit is for tools like:

- CV Reality Check AI;
- EdTech Funnel Audit;
- Lead Handoff Audit;
- CRM Health Check;
- Offer Clarity Checker;
- SaaS Stack Reality Check.

## Default MVP Shape

```text
Hero
Form
Loading state
Result page
Copy / download / share
Lead CTA
Privacy note
Basic analytics
```

## Default Build Order

1. Lovable builds the first UI and mock result.
2. Codex wires real API/database.
3. Codex polishes mobile, errors, analytics, privacy, and result quality.
4. Publish.
5. Send to 3-10 people first.

## Do Not Include In V1

- auth;
- payment;
- team accounts;
- complex admin;
- real integrations;
- full dashboard;
- browser extension;
- mobile app;
- advanced PDF unless simple download already works.
